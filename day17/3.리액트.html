리액트 개발환경 구축하는 방법! 1. node.js 설치 2. vscode가 켜져있었다면 종료후 재실행 3. 터미널(ctrl + `(백틱)) 열어서 node -v 입력(띄어쓰기 주의) 4. v19.0.0(현재버전에따라 다를수있음) 나오면 ok 5. dir을 입력해서 현재 있는 폴더의 자식 폴더들을 확인할 수 있다. 6. "cd 폴더명"을 입력해 원하는 폴더로 들어간다. 상위경로로 갈려면 "cd .." 입력하면 됨 7. npx create-react-app (프로젝트이름) 입력. 띄어쓰기 주의! 8. ok to proceed에 y입력후 엔터 9.리액트 실행은 해당 폴더로 들어가 npm start로 시작할수있다!
